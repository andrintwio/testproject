tw_infrastructure is a base module that provides the foundation for infrastructure management in Odoo. It establishes the Infrastructure menu, security groups, and settings section that other infrastructure-related modules can extend.

What the Module Does

This module creates the infrastructure management framework in Odoo by:

  • Adding an Infrastructure menu item to the main navigation
  • Creating security groups for infrastructure management (Admin, Consultant, Developer)
  • Adding an Infrastructure section in Settings where infrastructure-related modules can add their configuration options

This module serves as a base for other infrastructure modules and does not provide standalone functionality on its own.

Why the Module is Useful

This module provides the foundation for infrastructure management by:

  • Centralized Infrastructure Management: Creates a dedicated menu and settings area for all infrastructure-related tools
  • Security Framework: Establishes security groups that can be used across infrastructure modules for consistent access control
  • Extensibility: Allows other modules to extend the Infrastructure menu and settings without duplicating code
  • Organization: Keeps infrastructure tools organized in one place, separate from other business applications

How Users Can Get Started

Initial Configuration

After installing this module:

  1. The Infrastructure menu will appear in the main navigation for users with appropriate permissions
  2. An Infrastructure section will be added to Settings (accessible via Settings > Infrastructure)
  3. Security groups will be available for assignment to users:
    • Infrastructure Admin: Full access to infrastructure management
    • Infrastructure Developer: Read and write access to infrastructure data
    • Infrastructure Consultant: Read-only access to infrastructure data
Setup Master Data

This module does not require any master data setup. It only provides the framework for other infrastructure modules.

To use this module effectively:

  1. Assign appropriate security groups to users who need access to infrastructure features
  2. Install infrastructure-related modules that extend this base module
Key Workflows

This module does not provide specific workflows. It serves as a foundation for other modules that will add workflows and functionality.

Once other infrastructure modules are installed, you can:

  • Access infrastructure features through the Infrastructure menu
  • Configure infrastructure settings in Settings > Infrastructure

Technical Details

List of New Fields and Models

This module does not add any new models or fields that are visible to end users. It only provides:

  • Security Groups: Available in Settings > Users & Companies > Groups
    • Infrastructure Admin
    • Infrastructure Developer
    • Infrastructure Consultant
  • Menu Item: Infrastructure menu in main navigation
  • Settings Section: Infrastructure section in Settings
Added Actions and Buttons

This module does not add any actions or buttons. It provides the framework for other modules to add their actions and buttons.
